# Net-Scanner
<p>A python script that allows you to perform a network scan on Linux computers.</p>

### Download :
```shell
sudo su
git clone https://github.com/Bqrayvzdgn/Net-scanner
cd ./Net-scanner
python net-scanner.py -h
```

### Usage :
```shell
sudo su
python mac-changer.py -i [IP Field]
```
